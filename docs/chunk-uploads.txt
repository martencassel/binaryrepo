Start upload
POST /name/blobs/uploads
Create empty file with {uuid} in /tmp/filestore/uploads/{uuid}
Return {uuid}

Upload chunk
POST /name/blobs/uploads/{uuid}
Headers: 
Content-Length = <size-of-chunk>
Content-Range = 0-0 (StartOfRange-EndOfRange)

1. Parse start := (StartOfRange)
2. Open uuid file in the uploads folder
3. Seek to the offset start from the beginning
4. Write bytes to offset
5. Set Range Header. Range = file.Position-1
Range: bytes=0-<offset>
